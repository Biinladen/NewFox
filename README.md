# NewFox
A minimal Firefox theme with userContent and userChrome

Internal Debugging =

in about:config make these true =

devtools.chrome.enabled

devtools.debugger.remote-enabled

then press =

Ctrl + Shift + Alt + I (Windows/Linux)

Cmd + Opt + Shift + I (Mac)

##############################################################################

Enabling the customizations files =

in about:config make these true =

privacy.userContext.enabled

toolkit.legacyUserProfileCustomizations.stylesheets

type = about:support
then profile folder then paste the chrome folder completly

##############################################################################

enabling weather =
in about:config make these this =

browser.newtabpage.activity-stream.showWeather

note =

disable personalize-button in the configs so the pencil icon get shown up in the home screen so the weather widget can be toggled-on , then enable the file again
